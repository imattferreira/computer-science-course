Criptografia

Criptografia Simétrica e Assimétrica

Criptografia é uma forma de proteger informações sensíveis onde somente a pessoa dona de uma chave criptográfica (similar a uma senha) específica consegue vê-las. Essas informações são ocultadas a partir de uma série de técnicas na qual elas se tornam ilegiveis para qualquer um que não tenha a chave. Existem dois tipos de chave criptográfica: a Chave Simétrica e a Chave Assimétrica. Para saber mais, teste criptografar algo escrevendo no campo logo abaixo.

Dica:

Para o texto poder ser criptografado corretamente, ele não pode conter:

  • Números (como 1, 2 e 3)
  • Caracteres especiais (como ç, ~, / e ã)
